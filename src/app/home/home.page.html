Motivação Esse é um projeto desenvolvido pelos alunos da faculdade UNIVESP, turma que iniciou no ano de 2020. O tema do Projeto Integrador (PI) neste semestre era Desenvolver um software com framework web ou aplicativo que utilize banco de dados, inclua script web (Javascript), nuvem, acessibilidade, controle de versão, integração contínua e testes. Incluir um dos seguintes requisitos: uso e fornecimento de API, análises de dados e IoT. O grupo optou por desenvolver um aplicativo híbrido para ANDROID E IOS utilizando o framework IONIC e programando com ANGULAR. Esse aplicativo tem o objetivo de calcular o tempo restante para Aposentadoria dos professores do Estado de São Paulo, necessidade que foi levantada após pesquisa que mostrou várias dúvidas após a alteração das regras de aposentadoria para servidores públicos.
